# 2026-MCM-Analysis-on-Dancing-with-the-Stars-

This project was completed for the 2026 Mathematical Contest in Modeling (MCM/ICM), where our team received the Honorable Mention (H Award).

The project investigates the voting dynamics of the television competition Dancing with the Stars (DWTS) and explores how judges' scores and audience votes jointly influence contestant outcomes.

Using historical competition data, we developed statistical and simulation-based models to:

Estimate hidden audience voting behavior
Identify factors affecting judges' scores and fan votes
Evaluate fairness of existing voting systems
Propose an improved voting mechanism balancing technical performance and audience engagement
